== Description ==

Category View Row Action is a simple plugin which adds a 'View' link for your Categories and Tags in the admin section so that you can quickly view posts belonging tothe category on the public site.

== Installation ==

1. Upload the `category_view_row_action` folder into your plugin directory.
1. Activate the plugin through the 'Plugins' menu in WordPress

== Screenshots ==

1. Easily view posts belonging to a category

== Why I Created It ==

1. I work with many categories and needed a way to view them without visiting the public site
1. This works great with custom taxonomies in WordPress v3
